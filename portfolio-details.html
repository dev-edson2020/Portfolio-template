Projeto desenvolvido em React

Aplicação desenvolvida em em React para amantes de cinema que desejam descobrir os 10 melhores filmes do momento. Nesta aplicação, os usuários podem explorar uma lista de filmes emocionantes e, ao clicar em cada um, obter detalhes como sinopse, elenco e classificação. A interface responsiva e atraente proporcionam uma experiência intuitiva, enquanto a utilização de uma API externa, como o TMDb (The Movie Database), garante a atualização constante das informações.
